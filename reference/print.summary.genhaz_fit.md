# Print a summary of a fitted GH model

Print a summary of a fitted GH model

## Usage

``` r
# S3 method for class 'summary.genhaz_fit'
print(x, digits = 4, ...)
```

## Arguments

- x:

  A `"summary.genhaz_fit"` object from
  [`summary.genhaz_fit()`](https://aaronjehle.github.io/genhaz/reference/summary.genhaz_fit.md).

- digits:

  Number of significant digits. Default 4.

- ...:

  Currently unused.

## Value

Invisibly returns `x`.

## Examples

``` r
if (FALSE) { # \dontrun{
print(summary(fit))
} # }
```
