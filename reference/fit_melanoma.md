# Pre-computed melanoma GH model fit

A fitted `genhaz_fit` object from a generalised hazard model on the
[`biostat3::melanoma`](https://rdrr.io/pkg/biostat3/man/melanoma.html)
dataset (7,775 patients). See `data-raw/prepare_melanoma_fit.R` for the
fitting code.

## Usage

``` r
fit_melanoma
```

## Format

A `genhaz_fit` object; see
[`fit_genhaz`](https://aaronjehle.github.io/genhaz/reference/fit_genhaz.md).

## Source

[`biostat3::melanoma`](https://rdrr.io/pkg/biostat3/man/melanoma.html);
fit produced by `data-raw/prepare_melanoma_fit.R`.
